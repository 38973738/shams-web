Python Script — Number Guessing Game

PythonProgramming

A Python program that picks a random number and asks the user to guess it, giving higher/lower hints and counting the attempts. Small as it is, it uses loops, conditionals, variables, random number generation and user input — all the building blocks of real programming.

Why included: Programming is at the core of most IT roles. Even a short script shows I understand the logic involved in writing software, not just using it.

# Python Number Guessing Game
# Written for Unit 2 BTEC ICT

import random

def guessing_game():
  secret = random.randint(1, 100)
  attempts = 0
  while True:
    guess = int(input("Guess: "))
    attempts += 1
    if guess < secret: print("Too low!")
    elif guess > secret: print("Too high!")
    else: print(f"Got it in {attempts} attempts!"); break
